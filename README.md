垃圾项目，只为自己用起来顺手写的

usage:

        Doge-Obf.exe string
        
        Doge-Obf.exe string hi
        
        Doge-Obf.exe string 8   //0-8 Num
        
        Doge-Obf.exe str.txt
        
        Doge-Obf.exe str.txt hi
        
        Doge-Obf.exe str.txt 8
